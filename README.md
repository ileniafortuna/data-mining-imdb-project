# Data Mining Project – IMDb Dataset

## Overview
This project focuses on the analysis of an IMDb dataset containing information on movies, TV series and other media.

The work is divided into two main phases:
- foundational data mining techniques
- advanced analytical methods

## Dataset
The dataset contains information on thousands of titles (movies, series, etc.), described by multiple features such as ratings, votes, awards, genres and production details.

---

## Phase 1 – Data Mining Foundations

### Data Preparation
- Handling missing values
- Feature selection and reduction of redundant variables
- Outlier detection and transformation

### Clustering
- K-Means
- DBSCAN
- Hierarchical clustering

### Classification
- K-Nearest Neighbors
- Naive Bayes
- Decision Tree

### Regression
- Simple and multiple regression models

### Pattern Mining
- Extraction of frequent patterns from the dataset

---

## Phase 2 – Advanced Topics

### Outlier Detection
- Multiple methods (KNN, LOF, Isolation Forest, HBOS, etc.)
- Comparison and aggregation of results

### Imbalanced Learning
- Binary classification (award vs no award)
- Oversampling and undersampling techniques

### Advanced Classification
- Logistic Regression
- Support Vector Machines
- Neural Networks
- Random Forest
- Gradient Boosting

### Explainability
- LIME
- SHAP
- Counterfactual explanations

### Time Series Analysis
- Anomaly detection (local and global)
- Motifs and discords
- Clustering and classification of time series
